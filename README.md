# Multi-Platform Content Automation Workflow

This project is an n8n workflow that automates the process of turning a single content topic into platform-specific outputs for Instagram, Threads, and Whop.

## Workflow Architecture

![Workflow Architecture](multi-platform-content-workflow.png)

## Features

- Topic expansion and content briefing
- Hook generation
- Web research using Tavily API
- AI-based research summarization
- Script writing
- Platform-specific content generation
